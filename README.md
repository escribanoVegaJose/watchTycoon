# Watchmaker Tycoon

Juego 2D de gestion donde el jugador construye una marca de relojeria de lujo desde un taller pequeno hasta una maison internacional.

## Objetivo Inicial

Crear una primera experiencia movil con vista limpia de joyeria: empezar con dinero, comprar piezas en una subasta virtual por internet y poner las primeras piezas en venta.

## Estructura

- `scenes/`: escenas Godot separadas por area.
- `scripts/`: autoloads, gameplay, UI y utilidades.
- `data/`: datos editables del juego.
- `assets/`: placeholders, arte IA y recursos listos para Godot.
- `docs/`: diseno, economia, progresion y direccion artistica.
- `.opencode/agents/`: agentes especializados para desarrollo asistido.

## MVP

- 1 taller.
- 10 piezas.
- 4 materiales.
- 3 empleados.
- 5 pedidos.
- 1 reputacion global.
- 1 boton para pasar dia.
- 1 sistema de produccion.
- 1 sistema de entrega de pedidos.
