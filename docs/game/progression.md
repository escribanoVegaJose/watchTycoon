# Progresion

La progresion inicial se basa en reputacion global.

## Desbloqueos Iniciales

- Nuevas piezas.
- Mejores materiales.
- Pedidos con mayor recompensa.
- Empleados con mejores stats.

## Ritmo

La primera mejora debe llegar pronto para reforzar el loop principal.

## Segmentos de Marca de Referencia

Estos segmentos definen la progresión prevista del catálogo comercial. Todas las
marcas son ficticias y los rangos son precios de venta orientativos.

| Nivel | Marcas | Rango de precio |
|---|---|---:|
| Funcional | Nexora, Orvyn | 80–250 € |
| Clásico accesible | Montelune, Cadrien | 600–1.800 € |
| Premium | Vaudenne, Aurevant, Sorevain | 2.000–6.500 € |
| Lujo deportivo | Arctalon, Rovaryn, Nordanis | 7.000–22.000 € |
| Alta relojería | Maison Orselle, Aurelisse, Valcendre | 30.000–120.000 € |

El Salón de Lotes usa esta escala de precios. Las valoraciones de las piezas se
calculan a partir de calidad, mecanismo, prestigio de marca, estado, rareza,
demanda y coherencia del precio solicitado.
