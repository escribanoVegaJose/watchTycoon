# Economia

La economia debe ser simple para el MVP: coste de materiales + coste de produccion frente a recompensa del pedido o precio de venta.

## Variables Iniciales

- Dinero disponible.
- Coste de pieza.
- Coste de material.
- Dias de produccion.
- Recompensa del pedido.
- Bonificacion por calidad.

## Objetivo

El jugador debe entender rapidamente si un diseno es rentable y si mejora su reputacion.
